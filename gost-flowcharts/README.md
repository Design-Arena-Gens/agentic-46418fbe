## ГОСТ 19.701-90 · Нисходящие блок-схемы

Веб-приложение демонстрирует комплект согласованных блок-схем для программы, разбитой на функции, в соответствии с ГОСТ 19.701-90 (ISO 5807-85). Каждая функция визуализирована на самостоятельной диаграмме с соблюдением следующих принципов:

- строгий нисходящий поток без «петляющих» связей;
- обязательное слияние ветвей после каждого ромба;
- использование предопределённых процессов при вызове других функций.

### Запуск

```bash
npm install
npm run dev
```

Приложение будет доступно по адресу [http://localhost:3000](http://localhost:3000).

### Скрипты

- `npm run dev` — запуск окружения разработки;
- `npm run build` — сборка для продакшена;
- `npm run start` — запуск собранного приложения;
- `npm run lint` — проверка кода линтером.
